# Banco de Cuestiones Mejorado

Proyecto separado en archivos y con mejoras principales:

## Estructura
- `index.html`: estructura principal del sitio.
- `css/style.css`: estilos y mejoras visuales.
- `js/questions.js`: banco de preguntas de Neurociencias y Fisiología.
- `js/storage.js`: funciones auxiliares para `localStorage`.
- `js/simulado.js`: simulados aleatorios y cuaderno de errores.
- `js/app.js`: lógica principal del aplicativo.

## Mejoras agregadas
1. Panel de progreso general por área.
2. Cuaderno de errores automático.
3. Botón para revisar preguntas erradas.
4. Simulados aleatorios de 20, 50 y 100 preguntas.
5. Mejora visual del panel principal y tarjetas.
6. Separación del proyecto en varios archivos.

## Cómo subir a GitHub Pages
1. Descomprime este ZIP.
2. Sube `index.html`, la carpeta `css` y la carpeta `js` al repositorio.
3. Mantén los PDFs en la raíz del repositorio con los mismos nombres que ya usas:
   - `pdf_todas_questoes_neuroquiz_400_emmanuel.pdf`
   - `pdf_fisiologia_banco_completo_500_preguntas_emmanuel.pdf`
4. Espera que GitHub Pages actualice el sitio.
